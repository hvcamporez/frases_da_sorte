Sistema de frases da sorte feito em javascript.

No projeto 'Frases da Sorte', uma das funcionalidades é a exibição de mensagens conforme a resposta fornecida pelo usuário.

Ao iniciar o sistema, o usuário é solicitado a fornecer um número inteiro como resposta a uma pergunta. A resposta do usuário deve ser um número inteiro entre 1 e 31.

Se a resposta atender a condição definida (número entre 1 e 31), o sistema deve retornar uma frase da sorte referente ao número fornecido.

Porém, se a resposta do usuário for um número menor que 1 ou maior que 31, o sistema deve retornar a mensagem DIA INVÁLIDO.
